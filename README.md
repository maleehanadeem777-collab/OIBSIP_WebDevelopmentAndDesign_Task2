# OIBSIP-WebDevelopmentAndDesign-Task2
# 🌐 Authentic Developer Portfolio | Maleeha Nadeem

### 🚀 Project Overview
This is my official professional developer portfolio website designed and engineered from scratch. It serves as a centralized digital hub to showcase my technical skills, software engineering background, academic timeline, and ongoing internship projects with **Oasis Infobyte** and **Synent Technologies**. Built with a futuristic dark aesthetic, neon accents, and responsive layout guardrails.

---

## 🛠️ Key Features Implemented
* **Premium Dark Aesthetic:** Engineered using CSS custom properties with an immersive Slate and Electric Cyan (`#00f2fe`) neon glow design language.
* **Dynamic Interactive Navigation:** Fully functional sticky top navigation bar built with a crystal-clear backdrop blur matrix.
* **Professional Showcase Sections:**
  * **Hero Area:** Features direct CV download access and a glowing, border-animated modern profile frame.
  * **Services Grid:** Highlights core competencies in Frontend Engineering, UI/UX, and Fluid Responsive Architecture.
  * **Interactive Journey Timeline:** A customized vertical experience track displaying educational benchmarks and running internships.
  * **Project Vault:** Responsive card layouts showcasing technical stacks, custom badges, and live demonstration video tags.
* **Zero-Overlap Grid System:** Built entirely on CSS Grid and Flexbox to prevent breaking or component overlapping on standard viewport resizes.
* **Fully Validated Contact Matrix:** Clean and integrated communication form with interactive responsive focus inputs.

---

## ⚙️ Core Technical Stack
* **Markup Architecture:** Semantic HTML5 (`<header>`, `<main>`, `<section>`, `<footer>`) for modern accessibility and clean DOM crawling.
* **Styling Framework:** Vanilla CSS3 using custom global variables, strict CSS Grid configurations, linear Flexbox alignments, and interactive transitions.
* **Vector Elements:** FontAwesome v6.0 integration for streamlined structural iconography.
